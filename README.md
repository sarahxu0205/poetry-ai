# 诗中天地

> 与诗人共品唐诗的 AI 互动体验

穿越千年时光，与李白对饮、与杜甫论诗。在 AI 构建的唐诗世界中，感受诗人的灵魂与诗意。

## 项目简介

**诗中天地** 是一个基于 AI 的唐诗互动 Web 应用。用户可以选择多位唐代诗人，进入模拟的诗会场景，围绕经典唐诗展开讨论；也可以与诗人一对一交流，共同创作诗歌。每位诗人都拥有独特的性格、对话风格和系统提示词，让互动体验更加真实生动。

## 核心功能

### 诗会模式
选择 2-4 位诗人，邀请他们参加一场跨越千年的诗会。AI 诗人会围绕你选定的唐诗，从意境、格律、情感等不同角度展开讨论，你也可以随时加入对话，发表自己的见解。

### 创作模式
选择一位诗人作为你的创作伙伴，描述你想要的场景或主题，诗人会为你生成候选诗句。你可以点击"采纳"将喜欢的诗句加入自己的诗作，最终完成一首完整的诗。

### 与诗人聊天
在诗人详情页，你可以与任何一位诗人自由聊天，探讨诗歌、人生、理想、自然等话题。

### 诗人群像
浏览 10 位唐代诗人的生平、诗风、代表作品，了解他们的故事与创作背景。

### 我的创作
保存和管理你与 AI 诗人共同创作的诗作，随时回顾自己的诗意旅程。

## 技术栈

- **前端**：原生 HTML5 + CSS3 + JavaScript（ES6+）
- **AI 服务**：支持 DeepSeek API / 模拟模式（无需 API）
- **样式风格**：中国古典水墨风格
- **数据存储**：localStorage（本地保存创作记录）

## 快速开始

1. 将项目部署到任意静态 Web 服务器（如 Nginx、Apache、Vercel、GitHub Pages 等）
2. 或通过本地 HTTP 服务器访问（如 VS Code Live Server、`npx serve`、`python -m http.server` 等）
3. 打开页面后，点击右上角设置按钮，可配置 DeepSeek API Key 获得更智能的 AI 回复
4. 无 API 时自动使用模拟模式，同样可以体验完整功能

## 使用说明

### 开启一场诗会
1. 点击导航栏"诗会"
2. 选择一首想要讨论的唐诗
3. 邀请 2-4 位诗人（点击诗人卡片选择）
4. 选择讨论主题（综合赏析、意境分析、格律探讨等）
5. 点击"开启诗会"，等待诗人入场
6. 诗人开场讨论结束后，你可以发言或点击"继续讨论"

### 与诗人一起创作
1. 点击导航栏"创作"
2. 选择一位合作诗人
3. 选择诗歌体裁（五言绝句、七言律诗等）
4. 输入创作主题（可选）
5. 点击"开始创作"
6. 描述你想要的场景，诗人会为你生成候选诗句
7. 点击诗句旁的"采纳"按钮，将喜欢的诗句加入诗作
8. 完成后点击"完成创作"，诗作将自动保存

### 查看我的创作
1. 在首页点击"我的创作"按钮（需已有保存的诗作）
2. 或在创作完成后自动跳转
3. 可以查看、复制或删除已保存的诗作

## 特色亮点

- **中国古典水墨风格 UI**：采用宣纸、水墨、朱砂、青绿等传统色彩，营造古典雅致的视觉氛围
- **独特的诗人人格**：每位诗人拥有独立的性格特征、对话风格和系统提示词，互动体验真实生动
- **点击诗人卡片即可聊天**：在诗人群像页点击任意诗人卡片，进入详情后可一键发起对话
- **双模式 AI 支持**：支持 DeepSeek API 真实对话，也提供离线模拟模式，无需 API 即可使用
- **响应式设计**：适配桌面端和移动端，随时随地与诗人对话
- **本地数据存储**：创作记录保存在浏览器本地，保护隐私
